# 馨光 AI 设计灯光安装指导

本教程用于内测。当前推荐方式是：腾讯云免密终端安装基础环境，龙虾负责后续米家绑定、家庭选择和本地安装馨光 Skill。

第一阶段脚本版本：`2026-06-25.24`
第二阶段脚本版本：`2026-06-26.12`
Skill 版本：`3.0.1`

## 内测流程

1. 创建腾讯云轻量服务器，并选择龙虾应用模板。
2. 在腾讯云龙虾应用设置里填写 MiMo 订阅 Key。
3. 在腾讯云服务器页面点击「免密码登录」，进入终端后粘贴安装命令。
4. 安装完成后，回到龙虾发送「绑定米家账号」。
5. 如果账号下有多个家庭，选择馨光设备所在家庭。
6. 发送「安装馨光 Skill」。
7. 安装成功后，说灯光需求，并按现场情况回复“已变化”或“未变化”。

## 终端安装命令

```bash
curl -fsSL https://nijez.github.io/xingguang-ai-lighting-guide/install-xinguang-ai-light.sh -o install-xinguang-ai-light.sh && bash install-xinguang-ai-light.sh
```

如果终端窗口中断，可以在同一个终端运行：

```bash
bash install-xinguang-ai-light.sh status
```

## MiMo Key

进入腾讯云龙虾应用设置，填写 MiMo 订阅 Key。填写完成后，再进行终端安装。

获取 Key：
https://platform.xiaomimimo.com/console/plan-manage

## 龙虾后续操作

安装完成后，回到龙虾：

```text
绑定米家账号
```

绑定成功后，如果账号下有多个家庭，请选择馨光设备所在家庭。家庭选择完成后，发送「安装馨光 Skill」。

```text
安装馨光 Skill
```

## 灯光测试示例

```text
客厅来个马尔代夫的海边日落。
二楼客厅来一个森林晨光。
全屋灯带做一个朋友聚会氛围。
```

灯光请求发出后，龙虾会提示你观察现场。

- 如果已变化，回复：已变化
- 如果未变化，回复：未变化

测试成功后，如果当前效果满意，也可以说：保存当前灯光效果到快照 3。

## 备用方式

如果必须在龙虾对话里尝试安装，可以使用页面里的备用方式。但如果龙虾提示无法执行脚本、要求查看脚本、反复解释风险，或中途切换成英文说明，请改用腾讯云免密终端安装。
