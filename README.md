# 馨光 AI 设计灯光安装指导

本教程用于今晚内测。用户只需要按顺序完成下一步，不需要了解底层组件或排障细节。

`2026-06-25.10` 为当前第一阶段安装脚本版本。

## 内测 6 步

1. 在腾讯云小龙虾应用设置里填写 MiMo 订阅 Key。
2. 打开小龙虾对话窗口，发送一键安装指令。
3. 安装完成后，按小龙虾提示绑定米家账号。
4. 如果账号下有多个家庭，选择馨光设备所在家庭。
5. 家庭选择完成后，继续安装馨光 Skill。
6. 直接说灯光需求，并按现场情况回复“已变化”或“未变化”。

## MiMo Key

进入腾讯云小龙虾应用设置，填写 MiMo 订阅 Key。填写完成后，再打开小龙虾对话窗口发送安装指令。

获取 Key：
https://platform.xiaomimimo.com/console/plan-manage

## 灯光测试示例

```text
客厅设计一个马尔代夫灯光效果。
二楼客厅来一个森林晨光。
全屋灯带做一个朋友聚会氛围。
二楼客厅来个佛光普照。
```

灯光请求发出后，小龙虾会提示你观察现场。

- 如果已变化，回复：已变化
- 如果未变化，回复：未变化

测试成功后，如果当前效果满意，也可以说：保存当前灯光效果到快照 3。
