请执行「馨光 Skill 安装」。全程必须使用中文回复。

重要要求：
1. 不要重复执行第一阶段安装。
2. 不要让我选择产品类型、设备编号或设备型号。
3. 如果检测到多个米家家庭，请让我选择馨光设备所在家庭，不要自动使用第一个家庭。
4. 安装完成后，只提示我可以直接说灯光需求。
5. 不要展开复杂说明。
6. 不要控制灯光，本指令只安装 Skill、准备灯控服务、读取当前家庭设备列表。

请只执行下面这份固定流程：

```bash
TARGET=/tmp/install-xinguang-skill.sh
LOG=/tmp/xinguang-skill-install-current.log
STATE=/tmp/xinguang-skill-install.state
RUN_MARKER=/tmp/xinguang-skill-install.pid
EXPECTED_VERSION=2026-06-26.4

if [ -f "$RUN_MARKER" ]; then
  OLD_PROCESS="$(cat "$RUN_MARKER" 2>/dev/null || true)"
  if [ -n "$OLD_PROCESS" ] && ps -p "$OLD_PROCESS" >/dev/null 2>&1; then
    echo "正在安装馨光 Skill。"
    echo "稍后请发送：查看馨光 Skill 安装进度"
    exit 0
  fi
fi

rm -f "$TARGET"
for URL in \
  https://nijez.github.io/xingguang-ai-lighting-guide/install-xinguang-skill.sh \
  https://raw.githubusercontent.com/nijez/xingguang-ai-lighting-guide/main/install-xinguang-skill.sh \
  https://cdn.jsdelivr.net/gh/nijez/xingguang-ai-lighting-guide@main/install-xinguang-skill.sh
do
  if curl -fsSL "$URL" -o "$TARGET" && grep -q "XINGUANG_SKILL_INSTALLER_VERSION=\"$EXPECTED_VERSION\"" "$TARGET"; then
    break
  fi
done

grep -q "XINGUANG_SKILL_INSTALLER_VERSION=\"$EXPECTED_VERSION\"" "$TARGET" || { echo "馨光 Skill 安装脚本下载失败，请联系工作人员处理。"; exit 1; }
chmod +x "$TARGET"

nohup env \
  INSTALL_ACTION=continue \
  INSTALL_NONINTERACTIVE=1 \
  bash "$TARGET" >> "$LOG" 2>&1 &
echo $! > "$RUN_MARKER"
sleep 2

echo "正在安装馨光 Skill。"
echo "稍后请发送「查看馨光 Skill 安装进度」。"
```

执行后只按下面格式回复：

```text

```
