馨光 AI 设计灯光安装指导

安装交给腾讯云终端，对话交给龙虾，灯光设计交给馨光 Skill。

安装脚本版本：2026-06-25.31
推荐方式：在腾讯云服务器页面点击「免密码登录」，进入终端后粘贴安装命令。

项目目标

终端安装基础环境，并预置馨光 Skill 本地安装器。完成米家绑定和家庭选择后，发送「安装馨光 Skill」；安装成功后，你可以对龙虾说灯光需求，例如：“客厅来个马尔代夫的海边日落。”

灯光请求发出后，龙虾会让你现场确认灯光是否变化。

安装流程

1

准备龙虾服务器

创建腾讯云轻量服务器，选择龙虾应用模板。

2

填写 MiMo 订阅 Key，并绑定微信通道

在腾讯云龙虾应用设置里填写 Key，并完成微信授权接入。

3

腾讯云免密终端安装

点击「免密码登录」，在终端粘贴本页安装命令。

4

回到龙虾绑定米家账号

直接对龙虾说：绑定米家账号。龙虾给出授权链接后，点击完成授权，把授权码粘贴回来。

5

选择米家家庭

如果账号下有多个家庭，请选择馨光设备所在的家庭。

6

安装馨光 Skill

家庭选择完成后，直接对龙虾说：安装馨光 Skill。

7

测试灯光效果

馨光 Skill 安装成功后，再说灯光需求，并按现场情况回复“已变化”或“未变化”。

准备龙虾服务器

购买腾讯云轻量应用服务器时，选择龙虾（OpenClaw）应用模板。服务器创建完成后，先填写 MiMo 订阅 Key，并绑定微信通道，再进入免密终端安装。

购买腾讯云龙虾服务器

打开腾讯云活动页。

查看腾讯云 OpenClaw 实践教程

查看腾讯云官方实践教程。

填写 MiMo 订阅 Key，并绑定微信通道

进入腾讯云龙虾应用设置，填写 MiMo 订阅 Key；在「通道」区域选择「微信」，点击「授权接入」完成绑定。后续用户通过微信和龙虾对话实现灯光控制。

不要在龙虾对话里反复粘贴 MiMo 订阅 Key。Key 只需要在腾讯云龙虾应用设置里填写一次。

小米 MiMo 订阅管理 / 获取订阅 Key

进入 MiMo 控制台获取订阅 Key。
重点确认：通道区域选择「微信」并完成授权接入。后续用户通过微信和龙虾对话控制灯光；页面里的 AgentChat 用于网页端测试。
需要进入网页端测试时，在「通道」区域点击「使用 AgentChat 即刻开始对话」。进入 AgentChat 后，默认选择「通用助手」，点击「下一步」，再点击「开启对话」。

推荐方式：腾讯云免密终端安装

在腾讯云服务器页面点击「免密码登录」，进入终端后粘贴下面这条命令。安装完成后，回到龙虾对话窗口继续绑定米家账号。

curl -fsSL https://nijez.github.io/xingguang-ai-lighting-guide/install-xinguang-ai-light.sh -o install-xinguang-ai-light.sh && bash install-xinguang-ai-light.sh

如果终端窗口中断，可以在同一个终端运行状态查询命令。不要重复运行安装命令。

bash install-xinguang-ai-light.sh status
终端安装完成后，建议先到腾讯云「快照」页创建一次快照，后续测试需要回滚时更方便。
终端看到“安装完成”后，回到龙虾，发送「绑定米家账号」。

绑定米家账号

安装完成后，回到龙虾对话窗口，直接说：绑定米家账号。

绑定米家账号
  1. 龙虾给出米家授权链接。
  2. 点击链接打开米家授权页，点击「确认授权」。
  3. 授权成功后点击「复制授权信息」。
  4. 把授权码粘贴回龙虾。
  5. 绑定成功后，龙虾会检查家庭列表。

选择米家家庭

如果你的米家账号下有多个家庭，龙虾会列出家庭。请选择馨光设备所在家庭，例如“林坞店”。

screenshots/08-home-select.png
选择家庭后，发送「安装馨光 Skill」。

安装馨光 Skill

家庭选择完成后，直接对龙虾说：安装馨光 Skill。

安装馨光 Skill
安装成功后，龙虾会提示可以开始测试灯光。

测试灯光效果

馨光 Skill 安装成功后，再告诉龙虾想要的灯光效果。

客厅来个马尔代夫的海边日落。
二楼客厅来一个森林晨光。
全屋灯带做一个朋友聚会氛围。

龙虾发出灯光请求后，只需要现场观察灯光是否变化。

如果已变化，回复：已变化。如果未变化，回复：未变化。

测试成功后，如果当前效果满意，也可以说：保存当前灯光效果到快照 3。

常见问题

终端安装卡住怎么办？

如果长时间没有新提示，运行终端状态查询命令；不要重复运行安装命令。

MiMo 订阅 Key 在哪里填写？

在腾讯云龙虾应用设置里填写。

米家账号怎么绑定？

按龙虾给出的授权链接完成授权，把授权码粘贴回龙虾。

有多个米家家庭怎么办？

选择馨光设备所在的家庭。

灯光未变化怎么办？

回复「未变化」。龙虾会提示测试未通过，请联系工作人员处理。
