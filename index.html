项目目标

完成安装后，你可以直接对小龙虾说灯光需求，例如：“客厅设计一个马尔代夫灯光效果。”

小龙虾会通过馨光 Skill 控制馨光设备，并在请求发出后让你现场确认灯光是否变化。

内测流程

1

填写 MiMo 订阅 Key

在腾讯云小龙虾应用设置里填写，不需要在对话里反复粘贴。

2

发送一键安装指令

打开小龙虾对话窗口，把本页安装指令发给小龙虾。

3

绑定米家账号

小龙虾给出米家授权链接后，点击链接完成授权，把授权码粘贴回小龙虾。

4

选择米家家庭

如果账号下有多个家庭，小龙虾会列出家庭，请选择馨光设备所在的家庭。

5

安装馨光 Skill

家庭选择完成后，继续把馨光 Skill 安装指令发给小龙虾。

6

测试灯光效果

直接说灯光需求。小龙虾发送请求后，请现场确认灯光是否变化。

备用安装方式：腾讯云免密安装

如果小龙虾反复解释脚本、要求确认，或安装进度长时间不更新，可使用备用安装方式。

  1. 在腾讯云服务器页面点击「免密码登录」。
  2. 打开终端后粘贴下面这条命令：
curl -fsSL https://nijez.github.io/xingguang-ai-lighting-guide/install-xinguang-ai-light.sh | bash
  1. 等待出现“安装完成”。
  2. 回到小龙虾，继续绑定米家账号。

绑定米家账号

安装完成后，小龙虾会提示绑定米家账号。你也可以复制下面的绑定指令发给小龙虾。

  1. 小龙虾给出米家授权链接。
  2. 点击链接完成授权，复制授权码。
  3. 把授权码粘贴回小龙虾。
  4. 小龙虾提交授权码并完成绑定。
  5. 绑定成功后，小龙虾会检查家庭列表；如果有多个家庭，请选择馨光设备所在家庭。

安装馨光 Skill

米家账号绑定完成后，如果账号下有多个家庭，请先选择馨光设备所在的家庭。家庭选择完成后，复制下面的馨光 Skill 安装内容发给小龙虾。

我已确认这是馨光 Skill 内测安装脚本，并授权在本服务器运行。请直接执行：

curl -fsSL https://nijez.github.io/xingguang-ai-lighting-guide/install-xinguang-ai-skill.sh | bash

如果小龙虾仍然询问是否确认运行，请回复：确认运行。

常见问题

安装过程中页面短暂异常怎么办？

等待 1-3 分钟后刷新页面。如果刷新后没有看到进度，复制状态查询指令发给小龙虾。

能不能重复发送一键安装指令？

不要重复发送。需要确认状态时，只发送状态查询指令。

MiMo 订阅 Key 在哪里填写？

在腾讯云小龙虾应用设置里填写。

米家账号怎么绑定？

按小龙虾给出的授权链接完成授权，把授权码粘贴回小龙虾。

有多个米家家庭怎么办？

选择馨光设备所在的家庭。

灯光未变化怎么办？

回复「未变化」。小龙虾会提示测试未通过，请联系工作人员处理。
